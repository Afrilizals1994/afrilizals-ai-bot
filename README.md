# 🤖 Afrilizals AI Bot (G4F Engine) v1.1

Bot Telegram cerdas berbasis Python yang menggunakan perpustakaan **G4F (GPT4Free)** untuk mengakses model AI tingkat tinggi tanpa memerlukan API Key berbayar.

## 🚀 Fitur Utama
* **No API Key Required**: Menggunakan *dynamic provider search* untuk mencari jalur AI gratis yang aktif.
* **Private Access**: Terkunci secara permanen hanya untuk merespons ID pengguna **8467632181**.
* **System Monitor**: Perintah `/status` untuk mengecek penggunaan RAM dan CPU perangkat secara real-time.
* **Remote Control**: Perintah `/off` untuk mematikan bot dari jarak jauh.

## 🛠 Teknologi
* **Language**: Python 3.x.
* **Libraries**: `pyTelegramBotAPI`, `g4f`, `requests`, `psutil`.
* **Engine**: Hybrid GPT-3.5/4 via G4F.

## ⚙️ Instalasi Lokal (Pydroid 3)
1.  Instal dependensi: 
    `pip install g4f pyTelegramBotAPI requests psutil`
2.  Set Environment Variable `TELEGRAM_BOT_TOKEN` di sistem.
3.  Jalankan `main.py`.
