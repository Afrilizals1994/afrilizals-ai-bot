# Afrilizals AI Bot (G4F Engine)

Bot Telegram cerdas berbasis Python yang menggunakan library **G4F (GPT4Free)** untuk akses model AI tingkat tinggi tanpa memerlukan API Key berbayar.

## 🚀 Fitur Utama
* **No API Key Required**: Menggunakan *dynamic provider search* untuk mencari jalur AI gratis yang aktif.
* **Private Access**: Terkunci secara permanen hanya untuk merespons ID pengguna **8467632181**.
* **24/7 Cloud Ready**: Dirancang untuk berjalan di PaaS seperti Render atau Koyeb.

## 🛠️ Teknologi
* **Language**: Python 3.x.
* **Libraries**: `pyTelegramBotAPI`, `g4f`, `requests`.
* **Engine**: Hybrid GPT-3.5/4 via G4F.

## ⚙️ Instalasi Lokal (Pydroid 3)
1. Instal dependensi: `pip install g4f pyTelegramBotAPI requests`.
2. Jalankan `main.py`.
